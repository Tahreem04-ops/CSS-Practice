
## CSS Properties Used

### 1. `font-family`

Used to change the style of text.
**Example:** `font-family: Arial, sans-serif;`

### 2. `background-color`

Used to set the background color of an element.
**Example:** `background-color: #f4f4f4;`

### 3. `color`

Used to change the text color.
**Example:** `color: darkblue;`

### 4. `text-align`

Used to align text horizontally.
**Example:** `text-align: center;`

### 5. `margin`

Used to add space outside an element.
**Example:** `margin: 20px;`

### 6. `padding`

Used to add space inside an element.
**Example:** `padding: 15px;`

### 7. `width`

Used to set the width of an element.
**Example:** `width: 300px;`

### 8. `height`

Used to set the height of an element.
**Example:** `height: 25px;`

### 9. `border`

Used to add a border around an element.
**Example:** `border: 2px solid #ccc;`

### 10. `border-radius`

Used to make corners rounded.
**Example:** `border-radius: 10px;`

### 11. `box-shadow`

Used to add shadow around an element.
**Example:** `box-shadow: 0 0 10px lightgray;`

### 12. `font-weight`

Used to make text bold.
**Example:** `font-weight: bold;`

### 13. `font-size`

Used to change the size of text.
**Example:** `font-size: 18px;`

### 14. `border-collapse`

Used in tables to merge borders into a single border.
**Example:** `border-collapse: collapse;`

### 15. `display`

Used to control how an element is displayed.
**Example:** `display: block;`

### 16. `cursor`

Used to change the mouse pointer style.
**Example:** `cursor: pointer;`

### 17. `text-decoration`

Used to add or remove text decoration such as underline.
**Example:** `text-decoration: none;`

### 18. `margin-top`

Used to add space at the top of an element.
**Example:** `margin-top: 20px;`

### 19. `margin-bottom`

Used to add space at the bottom of an element.
**Example:** `margin-bottom: 10px;`

### 20. `margin-left`

Used to add space on the left side of an element.
**Example:** `margin-left: 20px;`

### 21. `margin-right`

Used to add space on the right side of an element.
**Example:** `margin-right: 15px;`

### 22. `:hover`

Used to apply style when the mouse moves over an element.
**Example:** `button:hover { background-color: navy; }`

### 23. `border-left`

Used to add a border only on the left side.
**Example:** `border-left: 5px solid darkblue;`

## CSS Selectors Used

### `body`

Used to style the whole webpage.

### `h1`, `h2`, `h3`

Used to style headings.

### `p`

Used to style paragraphs.

### `form`

Used to style the form box.

### `input`, `select`, `textarea`

Used to style form fields.

### `button`

Used to style buttons.

### `img`

Used to style images.

### `table`, `th`, `td`

Used to style tables.

### `nav a`

Used to style links inside navigation.

### `.class-name`

Used to style a specific class.
